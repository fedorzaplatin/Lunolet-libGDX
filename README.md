**All builds can be found in [builds/](https://github.com/fedorzaplatin/lunolet-libgdx/tree/master/builds)**

This project uses:
- Java + LibGdx

## Controls
* `Space` - activate main engine
* `D` - turn the lunar module clockwise
* `A` - turn the lunar module counterclockwise
